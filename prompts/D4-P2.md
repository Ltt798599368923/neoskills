# D4-P2：起草一条消息

## Prompt
```
角色：你是一个专业沟通助手。
受众：一个要给教授写邮件的大学生。

输入：帮我给张教授写一封邮件，请求把 AI+X Elite 20 的作业延期 2 天提交。原因是网络不稳定，打不开 Claude，没法完成环境冒烟测试。

输出格式：
- 正式邮件格式（主题、称呼、正文、结尾）
- 礼貌简洁（150 字以内）
- 包含一个具体的新截止日期

验收标准：邮件要尊重对方，清楚说明情况，并提出合理的解决方案。
```

## Response
**主题：** AI+X Elite 20 作业延期申请 - 刘婷婷

尊敬的张教授：

您好！我是 AI+X Elite 20 的学生刘婷婷。写这封邮件是想申请将当前作业延期 2 天提交。

由于最近网络不稳定，我一直无法打开 Claude desktop 来完成要求的环境冒烟测试。我的工作文件夹已经建好了，网络恢复后马上就能继续。

请问是否可以延期到 [新截止日期] 提交？我明白跟上课程进度很重要，也会确保这种情况不再发生。

感谢您的理解！

此致
敬礼
刘婷婷
AI+X Elite 20 学生
