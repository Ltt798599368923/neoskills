# D4-P1: Summarize a PDF in My Major

## Prompt
```
Role: You are a computer science teaching assistant.
Audience: Undergraduate student learning data structures.

Input: Summarize the key concepts of a typical "Introduction to Binary Trees" chapter from a data structures textbook.

Output Shape: 
- 3-5 bullet points of core concepts
- 1-2 common use cases
- 1 potential exam question

Acceptance: Summary should be clear enough for a student to understand binary trees without reading the full chapter.
```

## Response
### Binary Trees - Key Concepts Summary

**Core Concepts:**
- A binary tree is a hierarchical data structure where each node has at most two children (left and right).
- Tree traversal methods: In-order (left-root-right), Pre-order (root-left-right), Post-order (left-right-root).
- Binary Search Tree (BST): A binary tree where left child < parent < right child, enabling O(log n) search on average.
- Tree height determines the worst-case time complexity for operations.

**Common Use Cases:**
- File system directory structures
- Expression parsing in compilers
- Database indexing (B-trees, a variant)

**Potential Exam Question:**
Given a binary search tree, write the in-order traversal and explain why the output is always sorted.
